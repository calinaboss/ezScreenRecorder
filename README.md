# EzScreenRecorder

EZ Screen Recorder is a simple desktop application built with Electron using HTML, CSS, and JavaScript.

The application provides a basic desktop structure that can run on windows, macos, and linux using web technologies.

## Installation:

- git clone https://github.com/calinaboss/ezScreenRecorder.git
  
- cd ezScreenRecorder
  
- npm install
  
- npm start (run in development mode)


and then build it: 
npm run build
The compiled output will be generated in the dist folder
